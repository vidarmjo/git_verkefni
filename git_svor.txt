"#Git spurningar og svör"
1. Hvað gera eftirfarandi Linux skipanir?
	cd Opnar möppu
	cd ~ Opnar möppu sem þú ert í C:/User/Viðar
	ls sýnir allar directory's
	pwd 
	mkdir býr til nýja möppu
2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone afritar innihaldið af möppunni og setur þau í aðra möppu
	git log *Segir hvað hver notandi hefur gert við kóða
	git status *sýnir allar þær gerðir af kóða sem hefur verið gert
	git diff *Sýnir þann mun sem hefur breyst 
	git checkout *Gefur því sem er nýtt branch
3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff  sýnir muninn á því sem hefur verið breytt
	b) git diff --staged Sýnir það sem þú hefur gert eða átt við í kóða
	c) git diff commit1 commit2 sýnir þann mun sem er á tveim völdum möppum
4. 	Hvað er útgáfustýring (e.version control)? 
		Sýnir hvernig forritið hefur breyst milli útgáfanna
5.	Hverjir eru helstu kostir við að nota GIT?
		Það er hægt að vinna saman og búa til verkefni sem að git sameinar síðan
